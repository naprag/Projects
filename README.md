## **Project 1**

### **Reading List in C**

This project aims to simulate a Reading List in C.

#### **Features:**

- **Add Book**: Add new books with title, author, and price.<br>
- **View List**: Display all books in the reading list.<br>
- **Search Book**: Find books by title or author.<br>
